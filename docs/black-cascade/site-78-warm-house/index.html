Наблюдательная база Фонда в Чаде (зона Сахеля), прикрытие — НКО по мониторингу засухи. Центр агентурной сети по аномальной активности в Северной Африке. ~80 информаторов: старейшины, рейнджеры, врачи местных общин.

В течение 6 часов 15.06.2024 Повстанцы Хаоса физически устранили 53 информатора в разных точках Сахеля. Никаких следов координации между точками. Никто из убитых не успел подать сигнал тревоги.

Главный вывод: список агентов был доступен только одному отделу Фонда. Утечка не «случайная». Значит, у Фонда есть внутренний предатель в отделе разведки — и до настоящего момента он не установлен.

Руководитель сети — Аманду Дембеле, выжил, переведён в режим частичного укрытия. Местонахождение известно только трём сотрудникам Фонда.
